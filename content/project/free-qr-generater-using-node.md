# QR Code generator using Node.js 
A QR code generator Application. Pass the link and Qr Code generate.

### Get Todo request 
```http://localhost:3000/```

### How to run this Project

- Install dependency

```npm i``` 

- Run the Project

```npm run dev```


## Packages

| Name             | Version                                                                |
| ----------------- | ------------------------------------------------------------------ |
| body-parser | ^1.20.2 |
| qrcode | ^1.5.3 |
| express | ~4.18.1 |
| nodemon | ^3.0.2 |
